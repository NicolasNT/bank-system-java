# Bank System Java

Simple banking system developed in Java using Object-Oriented Programming concepts.

## Features

- Create account
- Deposit
- Withdraw
- Transfer funds
- List accounts

## Concepts Practiced

- Classes and Objects
- Encapsulation
- ArrayList
- Methods
- Constructors
- OOP
- Java Collections

## Technologies

- Java 21
- IntelliJ IDEA
- Git
- GitHub
